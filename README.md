# WeatherAPI

Open Weather Map API

### Open Weather Map API Project

This small project was created to test the Open Weather API using Vanilla Javascript.

## Technologies Used:
* Fetch API
* Javascript
* CSS including animations

### V0: 
- Get user location (latitude, longitude)
